# ✨ Random Quote Generator

A modern and responsive Random Quote Generator built using React.js and Tailwind CSS.
The app displays a random inspirational quote each time the user opens the app or clicks the "New Quote" button.

## 🚀 Features

* Random quote generation
* Clean and minimal UI
* Responsive design
* Smooth hover animations
* Built with React Hooks
* Tailwind CSS styling

## 🛠️ Technologies Used

* React.js
* Tailwind CSS
* JavaScript (ES6)

## 📌 Functionality

* Shows a random quote on page load
* Displays quote text and author name
* Generates a new quote on button click

## 🎨 UI Highlights

* Gradient background
* Modern card design
* Interactive button animations
* Mobile-friendly layout

## ▶️ Run Locally

```bash
npm install
npm run dev
```
