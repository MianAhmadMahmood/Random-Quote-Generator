# ✨ Random Quote Generator

A modern and responsive Random Quote Generator built using React.js and Tailwind CSS.
The app displays a random inspirational quote every time the user opens the app or clicks the **"New Quote"** button.

---

## 🚀 Features

* Random quote generation
* Displays quote text and author name
* New Quote button functionality
* Clean and minimal UI
* Fully responsive design
* Smooth hover animations
* Built using React Hooks
* Styled with Tailwind CSS

---

## 🛠️ Technologies Used

* React.js
* Tailwind CSS
* JavaScript (ES6)
* Vite

---

## 📌 Functionality

* Shows a random quote on initial page load
* Generates a different quote on every button click
* Displays author name clearly
* Responsive layout for all devices

---

## 🎨 UI Highlights

* Modern gradient background
* Glassmorphism inspired card UI
* Interactive button hover effects
* Mobile-friendly interface
* Minimal and clean design

---

## 📂 Project Setup

Clone the repository:

```bash
git clone <your-repository-link>
```

Navigate into the project folder:

```bash
cd random-quote-generator
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```
